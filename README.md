Data Professional Survey Analysis  Dashboard

About the Project:

This project presents an interactive Power BI dashboard built using a survey dataset of data professionals. The goal of the project is to analyze real-world trends in salaries, job roles, programming languages, and work-life balance.

The dashboard transforms raw survey data into meaningful insights that are easy to understand and useful for students, job seekers, and professionals interested in the data field.

Project Highlights:

• Built an interactive dashboard using Power BI
• Compared average salaries across different countries
• Analyzed salary distribution based on job roles
• Identified the most popular programming languages
• Explored salary differences between male and female professionals
• Evaluated work-life balance and salary satisfaction (happiness index)
• Derived key insights from real-world survey data

Dataset Information:

The dataset includes responses from data professionals with the following attributes:

• Country
• Job Title
• Salary
• Gender
• Favorite Programming Language
• Work-Life Balance Rating
• Happiness with Salary

Tools and Technologies Used:

• Power BI – Data visualization and dashboard creation
• Microsoft Excel – Data cleaning and preprocessing
• DAX – Measures and calculations

Key Insights:

• Data Scientists have the highest average salaries among the roles analyzed
• Python is the most widely used programming language
• Salaries vary significantly across countries
• There is a slight variation in salaries between genders
• Work-life balance satisfaction is moderate, while salary satisfaction is relatively lower
