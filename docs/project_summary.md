# Project Summary

This project analyzes survey data of data professionals using Power BI.

## Objective
To understand trends in:
• Salaries

• Job roles

• Programming languages

• Work-life balance


## Outcome
The dashboard provides clear insights that help:

• Students choose career paths

• Professionals understand industry trends

## Learnings
• Data cleaning using Excel
• Data visualization using Power BI
• Creating interactive dashboards
